# Data Cleaning and Preprocessing - Task 1

## Objective

To clean and preprocess a raw dataset by handling missing values, duplicates, inconsistent formats, and data type issues.

## Dataset

Customer Personality Analysis Dataset (Kaggle)

## Tools Used

* Microsoft Excel

## Data Cleaning Steps

1. Identified and handled missing values.
2. Removed duplicate records.
3. Standardized text formatting.
4. Converted dates into DD-MM-YYYY format.
5. Renamed column headers for consistency.
6. Verified data types and corrected formatting issues.
7. Saved the cleaned dataset for further analysis.

## Deliverables

* Original Dataset
* Cleaned Dataset
* Screenshots of cleaning process
* README Documentation
